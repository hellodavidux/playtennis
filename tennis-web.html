🎯 How to Play

▼
  • Your challenge: Build a connection chain from the starting player to the target player
  • Find intermediate players: You need to find exactly 6 players that create a chain
  • Follow the connection rules: Each step must use a specific connection type in order
  • Search and add players: Use the search boxes to find and add players to your chain
  • Complete the puzzle: When your chain is valid, you'll see success!

Connection Types:

🎾 Direct Opponents: Played against each other
🌍 Same Country: Both from the same nation
🤚 Same Hand: Both left-handed or right-handed
📏 Similar Height: Within 5cm of each other
📅 Same Birth Year: Born in the same year
🏆 Same Tournament: Played in the same tournament
